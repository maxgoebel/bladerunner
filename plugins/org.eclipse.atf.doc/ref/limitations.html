AJAX Toolkit Framework (ATF) Limitations

  • Debugger
    • Multiple simultaneous Mozilla debugger sessions are not supported. An error dialog will appear if more than one debug session is launched.
  • Intermittent crashes on Linux
    • We've had reports of intermittent problems bringing up Mozilla for the first time when installing under Linux. If this happens, try running it again -- the problem appears to resolve itself on the second try. We are investigating the problem.
